# RPG Elden Ring de Osasco

Um jogo de RPG, desenvolvido em Swift como projeto para o Apple Developer Academy.

# Sobre o Projeto

Este é um jogo de aventura interativo onde você é um personagem que tem que vancer osasco, escolha sua classe e enfrente criaturas em combates por turnos. O projeto foi desenvolvido aplicando conceitos fundamentais de programação em Swift aprendidos nas aulas do Workshop.

# Características

3 Classes Jogáveis: Guerreiro, Bandido, Mago ou Miseravel
Sistema de Combate por Turnos: ataques
Elementos Aleatórios: variação de dano e seleção de inimigos
